# Bulletin ICRSP — Dijon

Bulletin de la paroisse (Institut du Christ Roi Souverain Prêtre) à Dijon, adapté en page web.

## Contenu

- `index.html` — page principale du bulletin (version la plus récente). Sert de page d'accueil pour GitHub Pages.
- `bulletin-icrsp-dijon-mai-2026.html` — version archivée du bulletin de mai 2026.
- `bulletin-icrsp-dijon-mai-2026.pdf` — export PDF du bulletin de mai 2026.
- `icrsp_apostolats.kml` — carte des apostolats (ouvrable dans Google Earth / Maps).

## Publication

Pour publier en ligne : pousser ce dossier sur un dépôt GitHub, puis activer **GitHub Pages** (Settings → Pages → branche `main`, dossier `/root`). Le site sera servi à partir de `index.html`.
